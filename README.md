# Game Build Instruction

## Setting Up Engine
### Prerequisites
* CMake 3.15+
* C++17 compiler (Clang / GCC / MSVC)

### Steps
```bash
git clone https://github.com/yourusername/MyEngine.git
cd MyEngine
mkdir build && cd build
cmake ..
cmake --build . --config Debug
```
  
* Now, Go to the MyEngine/build/Debug folder and run MyEngine.exe

## Setting Up Game
* Download this repo.
* Open MyEngine 
* Click on Open Existing
* If this is the first time opening a project in the engine, you will need to stack the UI windows first.
