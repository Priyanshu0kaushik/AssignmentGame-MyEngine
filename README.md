# Game Build Instruction

## Setting Up Engine
### For Windows:
* First we have to clone https://github.com/Priyanshu0kaushik/MyEngineGameEngine repo.
* After downloading it, navigate to the MyEngine Folder. 
* Make a new folder named “build” and then open it from the terminal/cmd.
* Run “cmake ..” command.
* After it's completed, go into the build folder and open the MyEngine.sln in the visual studio.
* Then, in the Solution Explorer you will find the “MyEngine” project. Right Click on it and choose “Set up as Startup Module”.
* Now Build it using Ctrl + Shift + B and go to the MyEngine/Debug folder and run MyEngine.exe

## Setting Up Game
* Download this repo.
* Open MyEngine Click on OpenExisting
* Then Navigate to the folder where the AssignmentGame.meproject file is located and then press open.
* Now this main menu should open and you can now press Play on the top to play the game.
